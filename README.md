# Predictive Analytics Using Historical Data

## 📌 Project Overview
This project demonstrates predictive analytics using historical sales data. A Power BI dashboard is developed to analyze sales trends, identify business patterns, and forecast future sales using built-in forecasting features. The project helps organizations make data-driven decisions by visualizing historical performance and predicting future outcomes.

---

## 🎯 Objectives
- Analyze historical sales and revenue data.
- Identify sales trends over time.
- Forecast future sales using predictive analytics.
- Visualize business performance with interactive dashboards.
- Support better decision-making through data insights.

---

## 🛠️ Tools & Technologies
- Microsoft Excel
- Power BI Desktop
- Power Query
- Power BI Forecasting (Analytics Pane)
- Data Visualization

---

## 📊 Dataset Information
The dataset contains **100 historical sales records** with the following fields:

- Date
- Product
- Region
- Sales
- Revenue

---

## 📈 Dashboard Features

### KPI Cards
- Total Sales
- Total Revenue
- Average Sales
- Average Revenue

### Visualizations
- Sales Trend (Line Chart)
- Revenue Trend (Line Chart)
- Sales by Product (Bar Chart)
- Revenue by Region (Pie Chart)
- Monthly Sales Analysis (Column Chart)

### Forecast
- Future Sales Forecast using Power BI Analytics
- Trend Analysis
- Confidence Interval Visualization

### Filters (Slicers)
- Product
- Region
- Date

## 🚀 Steps to Run

1. Open **Power BI Desktop**.
2. Import the Excel dataset.
3. Create KPI cards and visualizations.
4. Add Product, Region, and Date slicers.
5. Use the **Analytics** pane to enable forecasting on the Sales Trend chart.
6. Save the dashboard as **Predictive_Analytics_Dashboard.pbix**.
